# 💼 Personal Portfolio Website

A modern and responsive **Portfolio Website** built using **HTML, CSS, and JavaScript** to showcase my projects, skills, and experience as a frontend developer.

## 🚀 Live Demo

🔗  https://abhay82k.github.io/Portfolio/

## ✨ Features

* 👨‍💻 About Me section
* 🛠️ Skills showcase
* 📂 Projects section (with live links)
* 📱 Fully responsive design
* 📧 Contact section
* 🎨 Clean and modern UI

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling
* **JavaScript** – Functionality

## 📂 Folder Structure

```bash
portfolio/
│── index.html
│── style.css
│── script.js
│── assets/
```

## ⚙️ How to Run Locally

1. Clone the repository

```bash
git clone https://github.com/your-username/portfolio.git
```

2. Open folder
3. Run `index.html` in browser
